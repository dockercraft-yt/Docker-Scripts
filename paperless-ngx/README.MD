## Script-Inhalt

```bash
#!/bin/bash

# === Laufzeit-Variablen ===
varRunTime=$(date +%Y%m%d_%H%M%S)
varDaysOlder=7

# === Verzeichnisse ===

} >> "$LOG_FILE" 2>&1
```
